  <action>
    Create `packages/config/banned-terms.json` with exactly:
      ["Part 141", "approved", "certified course"]

    Create `packages/config/eslint-rules/no-banned-terms.js` following the pattern in research §Pattern 6:
      - CommonJS module.exports (ESLint flat config accepts both but CJS is simplest for a local rule that Node can require directly from tests)
      - Reads banned-terms.json relative to __dirname
      - Compiles a case-insensitive word-boundary regex: `new RegExp('\\b(' + terms.map(escape).join('|') + ')\\b', 'i')`
      - meta: { type: 'problem', docs, schema: [], messages: { banned: 'Banned term "{{term}}" found. See packages/config/banned-terms.json. Add // allow-banned-term: <reason> above to allow.' } }
      - create(context): returns visitors for Literal, TemplateElement, JSXText
      - hasAllowComment(node): uses context.sourceCode.getCommentsBefore(node) (modern API; fallback to context.getSourceCode() for older eslint) and checks for /allow-banned-term:\s*\S/ in the comment value
      - check(node, value): if string, match regex, if match and !hasAllowComment(node) report({node, messageId:'banned', data:{term: m[1]}})

    Create `packages/config/eslint-rules/no-banned-terms.test.js` using Node's built-in `node:test` runner and ESLint's RuleTester (from 'eslint'). Cover the 7 behaviors above. Use `new RuleTester({ languageOptions: { parserOptions: { ecmaVersion: 2022, sourceType: 'module', ecmaFeatures: { jsx: true } } } })` for flat-config compatible testing. Invalid cases assert `errors: [{ messageId: 'banned', data: { term: '...' } }]`.

    Create `packages/config/eslint.config.js` (ESM flat config):
      ```
      import noBannedTerms from './eslint-rules/no-banned-terms.js';
      import tsParser from '@typescript-eslint/parser';
      import tsPlugin from '@typescript-eslint/eslint-plugin';
      import reactPlugin from 'eslint-plugin-react';
      import reactHooks from 'eslint-plugin-react-hooks';

      const part61Plugin = { rules: { 'no-banned-terms': noBannedTerms } };

      export default [
        { ignores: ['**/node_modules/**', '**/.next/**', '**/dist/**', '**/.turbo/**', '**/*.test.*', '**/*.spec.*', '.planning/**', 'supabase/**'] },
        {
          files: ['**/*.{ts,tsx,js,jsx}'],
          languageOptions: { parser: tsParser, parserOptions: { ecmaVersion: 2022, sourceType: 'module', ecmaFeatures: { jsx: true } } },
          plugins: { '@typescript-eslint': tsPlugin, 'react': reactPlugin, 'react-hooks': reactHooks, 'part61': part61Plugin },
          rules: {
            '@typescript-eslint/no-unused-vars': ['warn', { argsIgnorePattern: '^_' }],
            'react-hooks/rules-of-hooks': 'error',
          },
        },
        {
          files: ['apps/web/**/*.{ts,tsx,jsx}', 'apps/web/templates/**/*', 'packages/exports/**/*'],
          plugins: { 'part61': part61Plugin },
          rules: { 'part61/no-banned-terms': 'error' },
        },
      ];
      ```
