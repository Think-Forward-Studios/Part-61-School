    **1. adsb tRPC router** (`packages/api/src/routers/adsb.ts`):
    Create a router using `protectedProcedure` (all authenticated roles can view the fleet map):
    - `fleetPositions` query: Takes `{ bbox: BBox }` input. Reads all aircraft tail numbers from the school DB (scoped to school_id via `withTenantTx`), then calls `provider.getFleetPositions(tailNumbers, bbox)`. Also returns aircraft metadata (id, grounded_at) so the UI can determine status (grounded = red pin). Returns `{ fleet: EnrichedPosition[], feedHealthy: boolean }` where EnrichedPosition extends AircraftPosition with `aircraftId`, `tailNumber`, `isGrounded`, `activeReservationId`.
    - `traffic` query: Takes `{ bbox: BBox }` input. Calls `provider.getTrafficInBbox(bbox)`. Returns `{ traffic: AircraftPosition[] }`.
    - `flightTrack` query: Takes `{ tailNumber: string, minutes?: number }` input. Computes bbox from the school's active base lat/lon (with a generous 200nm / ~3.3 degree padding). Calls `provider.getFlightTrack(tailNumber, bbox, minutes)`. Also fetches the active reservation for this aircraft (if any) and returns its `route_string` for XC overlay. Returns `{ track: TrackPoint | null, plannedRoute: string | null }`.
    - `feedStats` query: Calls `provider.getStats()`. Returns FeedStats.
    - Instantiate `SwimAdsbProvider` with `process.env.ADSB_API_BASE_URL ?? 'http://localhost:3002'` at module scope (singleton).

    **2. admin.geofence tRPC router** (`packages/api/src/routers/admin/geofence.ts`):
    Create a router using `adminProcedure`:
    - `getActive` query: Returns the single active (non-deleted) geofence for the user's active base. Uses `withTenantTx` to enforce school_id scoping. Returns `Geofence | null`.
    - `upsert` mutation: Takes `{ kind: 'polygon' | 'circle', geometry: unknown, radiusNm?: number, label?: string }` input. Validates geometry is valid GeoJSON (basic structural check -- must have `type` and `coordinates`). If an active geofence exists for this base, soft-deletes it (sets `deleted_at = now()`), then inserts the new one. Returns the new Geofence. Wrapped in `withTenantTx`.
    - `softDelete` mutation: Takes `{ id: string }`. Sets `deleted_at = now()` on the geofence row, scoped by school_id + base_id. Returns `{ success: true }`.
