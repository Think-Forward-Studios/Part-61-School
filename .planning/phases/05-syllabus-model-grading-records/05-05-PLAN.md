      1. Header: school name, "Training Record" (NOT "Part 141 Record" — banned term), generated-at timestamp, page counter
      2. Student Identification: name, DOB, address, student cert number
      3. Course Identification: course title, rating sought, course_version label, enrollment/completion dates
      4. Chronological log: table of (date / subject / ground hrs / flight hrs / instructor name+cert / remarks) — one row per sealed grade sheet, ordered by conducted_at
      5. Test records: knowledge test, end-of-stage, practical (if any test_grades)
      6. Stage checks: table with date, stage, checker name + cert, pass/fail, remarks
      7. Endorsements issued: chronological list with rendered_text + signer snapshot
      8. Chief instructor attestation block — rendered only if enrollment.status='completed': "I certify that the student identified above has satisfactorily completed the training for {course.title}, meeting the objectives and completion standards of the course version {course_version.version_label}." + chief instructor signer snapshot (from enrollment.completed_by_user_id).

If lint flags it, move the citation into a data constant from the domain package where the lint glob does not apply. Use `@react-pdf/renderer` primitives. Signer snapshots render as `{full_name}, {certificate_type} {certificate_number}`. Letter size, Helvetica, narrow print-friendly margins, header and footer fixed per page.

    `iacra.pdf/route.ts` — Route Handler. Auth same pattern. Fetches student identification + `user_flight_log_totals` row for the user. Renders `IacraPdfDocument` → stream.

      - Student identification block
      - Totals table: Total time / PIC / SIC / Dual received / Solo / Cross-country (total / 50nm / with landings away) / Night / Night landings / Day landings / Instrument actual / Instrument simulated / Time in make/model (list)
      - Footer: generated-at, "This is a hours summary for IACRA 8710-1 data entry — not an official IACRA form."
    Banned-term clean.

    `iacra.csv/route.ts` — Route Handler returning `text/csv` with the same rows as the PDF, one column per category. First row headers. Filename `iacra-hours-{last_name}-{YYYYMMDD}.csv`.

    Wire the existing admin student record page (from 05-04) "Download 141.101 PDF" and "Download IACRA" buttons to these routes via `<a href>` tags (new tab).
